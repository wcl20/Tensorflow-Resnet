# Tensorflow-Resnet

## Setup
Generate Virtual environment
```bash
python3 -m venv ./venv
```
Enter environment
```bash
source venv/bin/activate
```
Install required libraries
```bash
pip install -r requirements.txt
```
Install Dataset [Tiny Imagenet](https://tiny-imagenet.herokuapp.com/)

Setup config file to dataset.

Build project
```bash
python3 build.py
```
Train model
```bash
python3 train.py
```
Test model
```bash
python3 test.py
```
